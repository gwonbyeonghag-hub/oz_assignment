🧮 콘솔 사칙연산 계산기

OZ 코딩스쿨 · JavaScript 1일차 과제 · 브라우저 콘솔에서 실행하는 계산기

  1. 개발자 도구를 엽니다 — F12 또는 ⌥⌘I(Mac) → Console 탭
  2. 페이지가 로드되면 자동으로 테스트 결과가 콘솔에 출력됩니다.
  3. 직접 계산하려면 콘솔에 아래처럼 입력하세요:
calculate("3 + 4 * 2")          // 11
calculate("10 - 2 * 3 + 8 / 4")   // 6
calculate("1 + 2 + 3 + 4 + 5 * 6") // 40

지원: 사칙연산 + - * / · 긴 수식 · 우선순위(*·/ 먼저). 소수점과 음수도 처리합니다. eval 없이 직접 구현했습니다.
